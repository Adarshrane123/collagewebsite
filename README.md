# Vivekananda Polytechnic Sitasaongi - Official Website

A fully responsive, modern website for Vivekananda Polytechnic Sitasaongi built with HTML, CSS, and JavaScript.

## 🌟 Features

### 🏠 **Home Page**
- Hero banner with animated heading
- College introduction and features
- Animated college logo with CSS keyframes
- Smooth scroll animations and hover effects

### 📚 **About Page**
- College history and establishment details
- Mission and vision statements
- Core values presentation
- Photo gallery with hover effects

### 👨‍🏫 **Faculty Page**
- Faculty member profiles with animated cards
- Department-wise filtering (Computer, Mechanical, Electrical, Civil)
- Contact information and qualifications
- Hover effects with photo zoom

### 📖 **Notes Page**
- Clean table layout for study materials
- Search functionality
- Department-wise filtering
- Download buttons for PDF notes
- Statistics and usage information

### 🤖 **AI Chatbot Page**
- Interactive chat interface
- Quick action buttons for common queries
- Simulated AI responses
- Typing indicators and animations

### 🔐 **Login Page**
- Student and Faculty login forms
- Input validation and error handling
- Password visibility toggle
- Responsive design for all devices

## 🔧 **Customization Guide**

### **Content Updates**
- **Text Content**: Edit the HTML files directly
- **Images**: Replace placeholder icons with actual images
- **Colors**: Modify CSS variables in `styles.css`
- **Faculty Data**: Update faculty information in `faculty.html`

### **Styling Changes**
- **Colors**: Update CSS custom properties in `:root`
- **Fonts**: Change Google Fonts imports in HTML files
- **Layouts**: Modify Bootstrap classes and CSS Grid/Flexbox

### **Functionality Modifications**
- **Chatbot Responses**: Edit the `generateBotResponse()` function in `script.js`
- **Form Validation**: Modify validation functions in `script.js`
- **Animations**: Adjust CSS animations and JavaScript timing
